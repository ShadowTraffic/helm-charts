## Charts

| Chart | Description |
|-------|-------------|
| [shadowtraffic](charts/shadowtraffic) | Runs ShadowTraffic as a Kubernetes Job |

## Usage

```bash
helm install my-release charts/shadowtraffic -f my-values.yaml
```

See each chart's README for full configuration details.
